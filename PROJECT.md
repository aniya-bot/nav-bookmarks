# PROJECT.md — 导航网页（收藏管理）

## 项目目标
做一个轻量导航网页，用来统一管理收藏链接，支持快速查找、分类整理和一键打开。

## 用户与场景
- 日常收藏文章/工具/视频
- 多分类管理（工作、学习、工具、资讯等）
- 关键字检索与标签筛选
- 快速打开高频网站

## MVP 范围（必须有）
- 新增收藏（URL、标题、分类、标签）
- 收藏列表/卡片展示
- 搜索与筛选（关键词、分类、标签）
- 编辑/删除收藏
- 新标签页打开链接
- 重复 URL 提示

## 后置功能（可二期）
- 登录与同步
- 导入导出（浏览器书签/CSV/JSON）
- 自动抓取标题/缩略图
- 批量操作、拖拽排序
- 访问统计与热度

## 技术路线（建议）
- 推荐：`Next.js + TypeScript + 本地 JSON/轻量存储`
- 理由：开发快、后续可扩展到登录同步与 API。

## 里程碑
- M1：信息架构与页面骨架
- M2：核心 CRUD + 搜索筛选
- M3：UI 优化 + 部署上线

## 验收标准
- 可新增/编辑/删除收藏
- 可按关键词、分类、标签检索
- 可稳定打开收藏链接
- 构建通过并可部署访问
